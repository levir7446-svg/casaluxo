💾 Salvamento automático

Toda vez que você adiciona, edita, exclui ou arrasta uma tarefa para outra etapa do funil, os dados são salvos automaticamente no armazenamento deste navegador (localStorage). Ao recarregar a página, eles continuam aqui.

📥 Exportar Backup (.json)

Gera um arquivo .json com todas as suas tarefas. Use para guardar uma cópia de segurança ou transferir os dados para outro computador/navegador.

📤 Importar Backup (.json)

Selecione um arquivo .json exportado anteriormente para restaurar as tarefas. Isso substitui os dados atuais.

📊 Exportar para Excel (.xlsx)

Gera uma planilha .xlsx com todas as tarefas, no mesmo formato da planilha "Controle de Leads". Ideal para abrir no Excel/Google Sheets ou enviar para alguém.

📈 Importar de Excel (.xlsx)

Selecione uma planilha .xlsx (com colunas Nome, WhatsApp, Produto, Status etc.) para importar as tarefas. Isso substitui os dados atuais.

🗑️ Limpar todos os dados

Remove todas as tarefas salvas neste navegador e reinicia o sistema do zero. Esta ação não pode ser desfeita — exporte um backup antes, se necessário.

📂 Importar Estoque via CSV

Faça upload de um arquivo .csv para atualizar o estoque em lote. Produtos existentes (pelo código) serão atualizados; novos serão adicionados.

📋 Formato do CSV (separado por ponto e vírgula):
Nº;Código;Categoria;Marca;Modelo;Dimensões;Valor (R$);Estoque
1;EU-001;ROUPEIRO MDF;EUROPA;Roupeiro 3 Portas;190x90x45cm;1299.90;8
ℹ️ A primeira linha (cabeçalho) é ignorada automaticamente.
🔑 O Código é a chave de correspondência para atualizar produtos existentes.
#CódigoModeloMarcaCategoriaEstoqueValorStatus

Nº | Código | Categoria | Marca | Modelo | Dimensões | Valor (R$) | Estoque
